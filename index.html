Contextualização

O projeto proposto tem como objetivo abordar uma questão crucial enfrentada pela sociedade contemporânea: a necessidade urgente de promover hábitos sustentáveis e proteger os oceanos. Vivemos em um mundo onde o consumo excessivo e a exploração irresponsável dos recursos naturais têm causado danos irreparáveis ao meio ambiente, impactando diretamente a saúde dos oceanos e ameaçando a vida marinha. Diante desse cenário, é fundamental encontrar maneiras eficazes de sensibilizar as pessoas e incentivá-las a adotarem práticas mais sustentáveis em seu dia a dia.

O aumento do consumo insustentável e a degradação dos oceanos representam desafios enormes para a sustentabilidade ambiental. A sociedade contemporânea está enfrentando uma crise ambiental sem precedentes, com a perda acelerada de biodiversidade, o aumento da poluição e as mudanças climáticas causando impactos devastadores em todo o mundo. No cerne dessa crise está o comportamento humano, impulsionado por padrões de consumo excessivos e uma mentalidade de curto prazo que prioriza o lucro imediato sobre a preservação do meio ambiente para as gerações futuras.

Além disso, a falta de conscientização e educação ambiental tem contribuído para a perpetuação desses padrões insustentáveis, com muitas pessoas desconhecendo o verdadeiro impacto de suas ações no meio ambiente. A ausência de incentivos tangíveis e recompensas para práticas sustentáveis também tem sido um obstáculo significativo, com muitas pessoas sentindo que suas pequenas ações individuais não fazem diferença em face dos enormes desafios globais.

Diante desse contexto, há uma necessidade premente de uma abordagem inovadora que engaje e motive as pessoas a adotarem hábitos mais sustentáveis e promova a proteção dos oceanos. É essencial criar ferramentas e plataformas que não apenas eduquem e conscientizem, mas também ofereçam incentivos tangíveis para as ações sustentáveis, capacitando os indivíduos a fazerem escolhas positivas para o planeta.

Descrição do projeto

A Blue Future será uma aplicação web gamificada projetada para incentivar hábitos de consumo sustentável e proteger os oceanos. Através de desafios diários, semanais e mensais, os usuários poderão acumular pontos, ganhar recompensas ecológicas e ver seu impacto positivo crescer. O site também oferecerá uma comunidade engajada, dicas, artigos e notícias sobre sustentabilidade, criando uma plataforma educacional e interativa para promover a conscientização ambiental.

Estrutura e funcionalidades:

Apresentação: É composta por três partes que apresentam as principais funcionalidades e a proposta do Blue Future.

Tela de login e cadastro: Interface onde os usuários podem se cadastrar e fazer login na aplicação.

Desafios: Seção onde os usuários podem acessar e participar de desafios ambientais.

Comunidade: Espaço onde os usuários podem interagir, compartilhar conquistas e aprender mais sobre sustentabilidade.

Dicas, notícias e artigos: Seção dedicada a fornecer informações e educar os usuários sobre práticas sustentáveis e questões ambientais.

Principais Módulos de Software

Front End da Aplicação Web

Linguagens de Programação: HTML, CSS, JavaScript - HTML, CSS e JavaScript são as linguagens fundamentais para o desenvolvimento web. HTML fornece a estrutura da página, CSS estiliza essa estrutura, e JavaScript adiciona interatividade e dinâmica, permitindo uma experiência de usuário rica e envolvente.

Framework Frontend: React.js - é um framework eficiente e flexível para a construção de interfaces de usuário. Ele permite a criação de componentes reutilizáveis, melhorando a produtividade e a consistência do código. Além disso, a biblioteca virtual DOM do React.js otimiza as atualizações da interface, proporcionando um desempenho superior.

IDE (Ambiente de Desenvolvimento Integrado): Visual Studio Code - é uma IDE poderosa e extensível, popular entre desenvolvedores web. Suporta uma ampla gama de extensões que aumentam a produtividade, incluindo depuradores, suporte a linting, e integração com sistemas de controle de versão.

Backend da Aplicação Web

Linguagem de Programação: Node.js - permite a construção de servidores escaláveis e de alto desempenho utilizando JavaScript. É especialmente eficiente para aplicações que exigem muitas operações de I/O, como manipulação de arquivos e comunicação em rede.

Framework Backend: Express.js - é um framework minimalista e flexível para Node.js. Ele facilita a criação de APIs e manuseio de requisições HTTP, permitindo um desenvolvimento rápido e eficiente do backend.

SGBD (Sistema de Gerenciamento de Banco de Dados): MongoDB - é um banco de dados NoSQL que oferece alta flexibilidade e escalabilidade. Ele é ideal para aplicações que requerem armazenamento de grandes volumes de dados não estruturados e mudanças frequentes no esquema de dados.

Autenticação e Segurança

Método de Autenticação: Login e cadastro de usuários, com criptografia de senha e verificação por e-mail - garante que os dados dos usuários estejam protegidos contra acesso não autorizado. A verificação por e-mail adiciona uma camada extra de segurança, confirmando a identidade dos usuários.

Integração com Serviços de Terceiros

APIs Externas: Integração com APIs de notícias ambientais - a integração com APIs externas permite que o aplicativo ofereça conteúdo atualizado e relevante, como notícias ambientais. Isso mantém os usuários informados e engajados com tópicos relacionados à sustentabilidade.

Armazenamento em Nuvem

Serviço de Armazenamento: AWS S3 (Amazon Simple Storage Service) - é um serviço de armazenamento de objetos em nuvem altamente escalável, seguro e durável. Ele é ideal para armazenar mídia, como imagens e vídeos, garantindo alta disponibilidade e proteção dos dados. O uso de AWS S3 permite fácil integração com outros serviços da AWS, como CloudFront para distribuição de conteúdo e Lambda para processamento de dados.

Comunicação em Tempo Real

Websockets: Utilização de Websockets para comunicação em tempo real entre usuários na comunidade - Websockets permitem comunicação bidirecional e em tempo real entre o servidor e o cliente. Isso é essencial para funcionalidades de chat e interações dinâmicas na comunidade do aplicativo, proporcionando uma experiência de usuário interativa e responsiva.

Acessibilidade

Semântica HTML adequada: Usar tags HTML semânticas ajuda na navegação por leitores de tela, melhorando a acessibilidade para usuários com deficiência visual.

Contraste de cores: Garantir bom contraste entre texto e fundo facilita a leitura para usuários com deficiência visual.

Tamanho do texto ajustável: Usar unidades de medida relativas permite que os usuários ajustem o tamanho do texto conforme necessário.

Legendas e alternativas de texto: Fornecer legendas para conteúdos multimídia e textos alternativos para imagens é vital para a acessibilidade de usuários com deficiência auditiva e visual.

Compatibilidade com leitores de tela: Testar com leitores de tela populares garante que o aplicativo seja acessível a usuários cegos ou com baixa visão.

Esses módulos de software compõem a arquitetura da solução para o desenvolvimento do projeto, fornecendo as bases necessárias para a construção de uma aplicação web robusta, segura e escalável. Cada componente foi escolhido com base na sua adequação às necessidades do projeto e na sua capacidade de fornecer uma experiência de usuário envolvente e interativa.
